🔐 Password Manager

A simple and secure password manager built with Python's `tkinter` for managing and storing passwords locally. It allows you to generate strong passwords and save login credentials easily.


✨ Features

* 🔑 Generate strong, random passwords
* 💾 Save credentials (Website, Email, Password) to a local file
* 📋 Easy-to-use graphical user interface (GUI)
* 🧠 Pre-filled email field with a default address
* ⚠️ Field validation to prevent empty entries
* ✅ Confirmation before saving credentials

* Python 3
* Tkinter (built-in GUI library)
* File handling for local storage (`data.txt`)


🚀 Getting Started


Prerequisites

* Python 3.x installed on your system

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/password-manager.git
   cd password-manager

2. Run the script:

python main.py


📁 Project Structure

password-manager/

* main.py           # Main application file
* data.txt          # File where passwords are saved (created after first save)
* logo.png          # Logo image displayed in the GUI
* README.md         # Project documentation


📝 Usage

1. Open the app.

2. Enter the website, your email/username, and either generate or enter your password.

3. Click Add to save the credentials locally to data.txt.
